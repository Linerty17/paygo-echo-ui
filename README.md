# PayGo Digital

PayGo is a digital financial services application for airtime, data purchases, and money transfers.

## Features

- User registration and authentication
- Dashboard with balance overview
- Airtime and data purchases
- Money transfers via PAY ID
- Account upgrades (Silver, Gold, Platinum, Emerald, Ruby, Diamond, Black)
- Weekly rewards system
- Live chat support

## Technologies

- React + TypeScript
- Vite
- Tailwind CSS
- Shadcn UI
- Lovable Cloud (Backend)
